
# 3D交换格式(exchange formats)

- STL:	Neutral
- OBJ:	ASCII variant is neutral, binary variant is proprietary
- FBX:	Proprietary
- COLLADA:	Neutral
- 3DS:	Proprietary
- IGES:	Neutral
- STEP:	Neutral
- VRML/X3D:	Neutral
